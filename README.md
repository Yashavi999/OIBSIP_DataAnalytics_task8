# Google Play Store Data Analysis

## Project Overview

This project involves analyzing **Google Play Store application data** to gain insights into app categories, user ratings, installation patterns, and pricing trends using data analysis techniques.

## Dataset

apps.csv

The dataset contains details about Android applications, including category, rating, number of installs, app size, price, and whether the app is free or paid.

## Objectives

* Examine and understand the dataset structure
* Perform data cleaning and preprocessing
* Study the distribution and popularity of app categories
* Visualize ratings and installation trends
* Compare patterns between free and paid applications

## Steps Performed

1. Loaded and explored the dataset
2. Cleaned the data and converted data types where necessary
3. Analyzed different application categories
4. Created visualizations for ratings and installation counts
5. Compared trends between free and paid apps

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Results

The analysis provided insights into widely used app categories, distribution of ratings, and patterns in application installations across the Play Store.

## Conclusion

This project highlights how data analysis techniques can be applied to understand trends in the mobile app marketplace and identify user preferences.
